How to View the AEA HTML Model
This folder contains the interactive HTML export of the Adapted Enterprise Architecture Artifact (AEA), modeled using Archi with NAF4 and ArchiMate views.

To open the model:
Download or clone the repository

Navigate to:
model/AEA_HTML_Export/

Open the file:
index.html

The model will open in your browser and allow you to explore:

L2 Organizational Architecture

L4 Process Architecture

L6 Activity-to-Role Mapping

L7 Information Flow

A8 Standards Mapping
